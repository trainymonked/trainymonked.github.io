Project Overview

Fancy Todo is not just to-do CRUD app. This is a schedule to-do app with a lot of features.

I actually use this website every day.

The features are, e.g.: adding, editing, marking as done, moving, deleting todos; custom categories, user authorization, dragging, keyboard navigation, use of reCAPTCHA, etc.

Feel free to check out the Project by visiting the Project Link.

Note that this is a ongoing pet-project. It might have some issues (not really responsive, PC only for now). Features to be added: password restore, responsive design, markdown in todos, dark mode.
